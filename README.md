## TwitterAPI を使用するための仮想環境を作成
```
pip install -r requirements.txt
source venv/bin/activate
```

## 環境変数設定

envファイルに以下を設定する
* API_KEY
* API_SECRET_KEY
* ACCESS_TOKEN
* ACCESS_TOKEN_SECRET
